Вступ

Система інвентаризації обладнання - це зручний інструмент для обліку, відстеження та управління пристроями в організації.

Основні можливості системи:

  • Облік пристроїв за типами, розташуванням і станом
  • Генерація унікальних інвентарних номерів
  • Друк інвентарних карток та QR-кодів
  • Історія змін пристроїв
  • Контроль доступу користувачів
  • Експорт та імпорт даних
  • Резервне копіювання та відновлення

Робота з пристроями

Додавання нового пристрою

Щоб додати новий пристрій:

  1. Натисніть кнопку "Додати пристрій" на сторінці зі списком пристроїв
  2. Заповніть форму, вказавши назву, тип, серійний номер та іншу інформацію
  3. Завантажте фотографії пристрою (опціонально)
  4. Натисніть "Зберегти"

Система автоматично згенерує унікальний інвентарний номер для пристрою.

Редагування пристрою

Для редагування інформації про пристрій:

  1. Відкрийте сторінку деталей пристрою
  2. Натисніть кнопку "Редагувати"
  3. Внесіть необхідні зміни
  4. Натисніть "Зберегти"

Усі зміни будуть збережені в історії змін пристрою.

Обслуговування пристроїв

Система дозволяє відстежувати обслуговування пристроїв:

  • Для кожного пристрою можна встановити дату останнього обслуговування
  • Налаштування інтервалу обслуговування (за замовчуванням 365 днів)
  • Система автоматично розраховує дату наступного обслуговування
  • Нагадування про необхідність обслуговування пристроїв
Як встановити дату обслуговування:
  1. При додаванні або редагуванні пристрою, вкажіть дату останнього обслуговування
  2. За потреби змініть інтервал обслуговування (кількість днів)
  3. Система автоматично розрахує дату наступного обслуговування
Регулярне обслуговування продовжує термін експлуатації пристроїв!

Інвентаризація

Система дозволяє проводити інвентаризацію пристроїв:

  • Друк інвентарних карток для окремих пристроїв
  • Масовий друк інвентарних карток для вибраних пристроїв
  • Фільтрація пристроїв за місцем розташування, типом та статусом
  • Експорт даних в Excel для подальшої обробки

QR-коди

Для кожного пристрою можна згенерувати QR-код, який містить основну інформацію про пристрій:

  1. Відкрийте сторінку деталей пристрою
  2. Натисніть кнопку "QR-код"
  3. Завантажте зображення QR-коду або відкрийте сторінку для друку

QR-коди можна наклеїти на пристрої для швидкої ідентифікації при інвентаризації.

Експорт/Імпорт

Експорт в Excel

Для експорту списку пристроїв в Excel:

  1. Перейдіть на сторінку зі списком пристроїв
  2. Натисніть кнопку "Експорт в Excel"
  3. Збережіть згенерований файл
Імпорт з Excel

Для імпорту пристроїв з Excel-файлу:

  1. Підготуйте файл Excel відповідно до шаблону
  2. Перейдіть на сторінку імпорту
  3. Завантажте файл
  4. Натисніть "Імпортувати"

Шаблон для імпорту можна завантажити зі сторінки імпорту.

Для адміністраторів

Адміністратори мають додаткові можливості:

  • Управління користувачами (додавання, редагування, блокування)
  • Управління містами
  • Перегляд історії дій користувачів
  • Налаштування резервного копіювання
  • Експорт та імпорт налаштувань системи

Резервне копіювання

Система підтримує автоматичне та ручне резервне копіювання:

Автоматичне резервне копіювання

Налаштування доступні в панелі адміністратора:

  • Вмикання/вимикання автоматичного резервного копіювання
  • Встановлення інтервалу (щоденно, щотижня, щомісяця)
  • Визначення терміну зберігання резервних копій
Ручне резервне копіювання

Адміністратори можуть створювати резервні копії вручну на сторінці резервного копіювання.

Відновлення з резервної копії

Для відновлення системи з резервної копії:

  1. Перейдіть на сторінку резервного копіювання
  2. Виберіть резервну копію
  3. Натисніть "Відновити"
Увага! Відновлення з резервної копії призведе до перезапису поточних даних!
